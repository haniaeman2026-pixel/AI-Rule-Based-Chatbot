
# AI-Rule-Based-Chatbot

##  Project Overview

AI Rule-Based Chatbot is a simple chatbot developed using Python. It responds to predefined user inputs using if-else statements and runs continuously until the user chooses to exit. This project demonstrates fundamental Artificial Intelligence concepts, control flow, and decision-making logic.

##  Features

* Responds to greetings such as Hello, Hi, and Assalam O Alaikum
* Answers basic predefined questions
* Handles exit commands like Bye, Exit, and Quit
* Uses if-else decision-making logic
* Runs in a continuous loop for user interaction
* User-friendly console interface

##  How to Run

1. Install Python on your computer.
2. Open the project in VS Code.
3. Open the terminal.
4. Run the following command:

python chatbot.py

5. Start chatting with the bot.

##  Sample Interaction

You: hello

Bot: Hello! Welcome to the AI Chatbot. How may I assist you today?

You: how are you

Bot: Thank you for asking. I am functioning perfectly and ready to assist you.

You: bye

Bot: Thank you for using AI Chatbot. Have a great day. Goodbye!
